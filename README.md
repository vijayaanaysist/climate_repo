# 🌍 Climate Change & Emissions Analytics Dashboard

## 🚀 Project Overview
This project is an end-to-end Machine Learning system that analyzes climate data, predicts climate risk levels (Low / Medium / High), and visualizes insights through an interactive Streamlit dashboard.

It also includes a Flask API for prediction and is deployable using Docker and Render.

---

## 🎯 Problem Statement
Climate data is:
- Scattered across multiple sources
- Difficult to analyze manually
- Lacking predictive insights

This project solves it by building a centralized AI-powered system.

---

## 🧠 Features

✔ CO₂ emissions analysis  
✔ Temperature trend analysis  
✔ Renewable energy impact  
✔ Precipitation handling (cleaned missing/zero values)  
✔ Climate risk prediction (ML model)  
✔ Interactive Streamlit dashboard  
✔ REST API using Flask  
✔ Docker container support  
✔ Postman API testing  
✔ Cloud deployment ready (Render)
